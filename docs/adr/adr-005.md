# ADR-005 Analysis Budget Defaults

- Status: accepted
- First needed: end of P2

## Context

Decompilation needs a default resource limit, but the pinned upstream callable
surface exposes instruction-count limiting rather than a wall-clock timeout.

## Decision

Apply a fixed per-request default of
`AnalysisBudget(max_instructions=100000)`. Callers may override
`max_instructions` explicitly. Do not expose a wall-clock timeout in this
contract.

## Consequences

- The default budget is predictable across the supported matrix.
- Callers can tune the instruction bound per request.
- Invalid or non-positive budget values remain hard input errors.
