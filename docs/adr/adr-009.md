# ADR-009 ISA Variant Scope

- Status: accepted
- First needed: end of P1

## Context

The project needed a clear boundary between broad bundled ISA availability and
the narrower set of ISA variants that flatline would actively back with
committed fixtures and confidence claims.

## Decision

Promise fixture-backed confidence for x86 32-bit and 64-bit, ARM64, RISC-V 64,
and MIPS32. Other ISA families and variants remain best-effort unless promoted
with dedicated coverage later.

## Consequences

- x86 gets both 32-bit and 64-bit committed fixture coverage.
- ARM32/Thumb, RV32, MIPS64, microMIPS, and other variants remain outside the
  fixture-backed promise.
- Support messaging must keep bundled availability separate from confidence
  level.
