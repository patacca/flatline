# Request & Result

[`DecompileRequest`][flatline.DecompileRequest] is the input payload for one
function decompilation.  [`DecompileResult`][flatline.DecompileResult] is the
output, carrying decompiled C code, structured function data, warnings, and
an optional error descriptor.
